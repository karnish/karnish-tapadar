Master's Thesis

Thesis Title: Personality Analysis from Social Media Data

The aim of this project was to automatically recognize personalities based on the Big-5 model based on users' status updates from Facebook using different supervised machine learning classifiers. We used 3 features, i.e. Count vectors, Tf-idf, and snowball stemmer along with tf-idf. We have used SVM, Multinomial Naive Bayes, KNN, and Logistic Regression for automatic recognition. Further, we have used ensemble modelling of the classifiers to find the best combination of the classifiers used for this work. Finally, we have tuned the vectorizers and parameters of the classifiers to improve their performance. For evaluating system performance, we used i)Accuracy and ROC-AUC, ii) Precision, Recall and F1 score, and iii) Misclassification error rate and False positive rate. In addition, we run 10-fold cross-validation and find cross-validated accuracy. Our demonstration shows that Multinomial Naive Bayes with stemmer plus Tf-idf feature attain the highest accuracy with the highest precision. From Ensemble modelling, we observed that the Logistic Regression and SVM blend achieve the highest accuracy and prove best for this personality analysis application.
